meggie.mainwindow.dynamic module¶

Contains functions that are used in the dynamic location and creation of tabs and datatypes. Can be used both internally and externally.

meggie.mainwindow.dynamic.construct_tab(source, package, tab_spec, parent)¶

Constructs analysis tab dynamically from python package containing an configuration file and code. Returns a QDialog that can be used within a QTabDialog of the main window.

Parameters

  • source (str) – Name of the module, should be meggie or meggie_*.

  • package (str) – Name of the tab, e.g. evoked or preprocessing.

  • tab_spec (dict) – The configuration.json of the tab read to a dict.

  • parent (instance of main window) – The main window, is passed to the handlers in the ui.py.

Returns

The constructed tab than can be added to main window.

Return type

instance of QDialog

meggie.mainwindow.mne_wrapper module¶

Wraps mne functions to log calls and parameters whenever called.

meggie.mainwindow.mne_wrapper.wrap(log_level, original_func)¶

Wraps a given function to log that the call is gonna happen before the call is made. Wrapping is done with a specific log level.

Parameters

  • log_level (str) – Should be debug, info, warning, error, etc.

  • original_func (function) – The function to be wrapped.

Returns

The wrapped function.

Return type

function

meggie.mainwindow.mne_wrapper.wrap_mne()¶

Goes recursively through mne package and wraps non-blacklisted functions so that they will get logged by meggie.

meggie.mainwindow.mne_wrapper.wrap_package(root, path, prefix)¶

A package wrapping function that can be called recursively.

Parameters

  • root (str) – Root of the package, e.g mne.preprocessing.

  • path (list) – A list containing the path to the package, e.g. [“/path_to_mne/preprocessing”].

  • prefix (str) – Path to the original root, e.g. “/path_to_mne”. This ensures that walk_packages does not walk out of the folder of interest due to a bug.

meggie.mainwindow.preferences module¶

Contains a class for handling reading and storing of global preferences such as the previous experiment and the tab settings. Normally saved to ~/.meggieprefs.
